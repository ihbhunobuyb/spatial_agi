# Research Skill - 通用研究调研流程

这个技能用于系统化地研究任何学术或技术主题，生成高质量的文献综述。

**核心特点**:
- ✅ 结构化调研流程
- ✅ 有述有评有总结
- ✅ 支持多种文献来源
- ✅ 生成规范化文档

---

## 📋 完整调研流程

### Step 1: 搜索文献 ✅

**搜索关键词策略**:
1. 核心关键词（中英文）
2. 相关领域关键词
3. 近义词和缩写
4. 知名作者/机构

**执行命令**:
```bash
cd ~/.openclaw/workspace/scripts

# 使用search_arxiv.py搜索
python3 search_arxiv.py "核心关键词1" --max 20
python3 search_arxiv.py "核心关键词2" --max 20
```

**其他来源**:
- Google Scholar
- PubMed
- Web of Science
- 领域专用数据库

---

### Step 2: 筛选文献 ✅

**筛选标准**:
1. **相关性**: 与主题直接相关
2. **创新性**: 有新方法或见解
3. **影响力**: 高引用/知名机构
4. **时效性**: 优先近期发表
5. **可获取**: 能获取到全文或摘要

**筛选数量**:
- 综述类: 20-30篇
- 研究类: 10-15篇

---

### Step 3: 深度分析 ✅

**核心问题（3个必问）**:

1. **是什么（What）**: 这篇研究的核心内容是什么？解决了什么问题？

2. **为什么（Why）**: 为什么会选择这个方向？有什么创新点？

3. **怎么做（How）**: 使用了什么方法？有什么技术路线？

**扩展问题**:
- 实验结果如何？
- 与其他方法相比有什么优劣？
- 有什么局限性？
- 对本研究方向有什么启发？

**分析工具**:
- NotebookLM（深度分析）
- 手动阅读摘要
- AI辅助总结

---

### Step 4: 整理综述 ✅

**文档结构模板**:

```
# [研究主题]文献综述

## 摘要
[简短概述研究领域和本文内容]

## 1. 引言
### 1.1 研究背景
### 1.2 研究意义

## 2. [主题1]
### 2.1 基本概念
### 2.2 研究进展
### 2.3 述评

## 3. [主题2]
### 3.1 基本概念
### 3.2 研究进展
### 3.3 述评

## 4. [主题3]
...

## 5. 相互作用机制/综合分析
[各主题之间的关系]

## 6. 临床/应用相关性
[实际应用场景]

## 7. 总结与展望
### 7.1 主要结论
### 7.2 当前挑战
### 7.3 未来方向

## 参考文献
```

## 📝 述评写作指南

### 什么是"述"？
- 客观描述研究成果
- 总结核心观点和方法
- 不带个人评价

### 什么是"评"？
- 分析研究的优缺点
- 评价创新性和局限性
- 提出自己的见解

### 什么是"总结"？
- 归纳主要结论
- 指出未来方向
- 提出建议

## 🔧 常用命令

### 搜索arXiv
```bash
cd ~/.openclaw/workspace/scripts
python3 search_arxiv.py "关键词" --max 20
python3 search_arxiv.py "关键词" --sort date
```

### 创建研究目录
```bash
mkdir -p ~/research/[主题]
```

### 常用搜索技巧
- 使用引号精确搜索: "exact phrase"
- 使用AND/OR: "A AND B"
- 排除关键词: "A NOT B"
