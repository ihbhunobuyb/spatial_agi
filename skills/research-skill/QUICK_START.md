# Research Skill - 快速开始指南

## 🎯 目标

对任意研究主题进行系统化调研，生成"有述有评有总结"的文献综述。

---

## 📋 快速流程

### Step 1: 搜索

```bash
cd ~/.openclaw/workspace/scripts

# 搜索相关论文
python3 search_arxiv.py "你的研究主题关键词" --max 20
```

### Step 2: 筛选

从搜索结果中筛选10-15篇核心文献。

### Step 3: 分析

对每篇核心文献回答3个问题：
1. **是什么**: 核心内容是什么？
2. **为什么**: 创新点在哪里？
3. **怎么做**: 使用了什么方法？

### Step 4: 撰写

按模板结构撰写综述：
- 述：客观描述
- 评：分析评价
- 总结：归纳展望

---

## 📁 输出位置

```
~/research/[主题]/
├── review.md      # 完整综述
└── summary.md    # 简报
```

---

## 📝 综述结构

```
# [主题]文献综述

## 摘要

## 1. 引言
## 2. [主题1] - 述+评
## 3. [主题2] - 述+评
## 4. [主题3] - 述+评
## 5. 相互作用机制
## 6. 总结与展望
```
