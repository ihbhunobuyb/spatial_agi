# Research Skill - 执行检查清单

## 调研前

- [ ] 明确研究主题
- [ ] 制定搜索策略（关键词列表）
- [ ] 确定输出目录

## 调研中

### 搜索阶段
- [ ] 使用至少3组关键词搜索
- [ ] 记录搜索结果
- [ ] 初步筛选相关文献

### 筛选阶段
- [ ] 按相关性排序
- [ ] 选择10-15篇核心文献
- [ ] 记录文献基本信息

### 分析阶段
- [ ] 每篇核心文献分析3个核心问题
- [ ] 记录关键发现
- [ ] 标注创新点和局限性

### 撰写阶段

#### "述"的检查
- [ ] 客观描述每个研究内容
- [ ] 准确引用核心观点
- [ ] 不带个人评价

#### "评"的检查
- [ ] 分析创新性
- [ ] 评价方法和结果
- [ ] 指出局限性
- [ ] 提出个人见解

#### "总结"的检查
- [ ] 归纳主要结论
- [ ] 指出当前挑战
- [ ] 展望未来方向
- [ ] 提出建议

## 调研后

- [ ] 检查文档结构完整性
- [ ] 格式规范检查
- [ ] 语法和错别字检查
- [ ] 导出最终版本

## 质量标准

| 项目 | 要求 |
|------|------|
| 文献数量 | 10-15篇核心文献 |
| 结构 | 清晰完整 |
| 内容 | 有述有评有总结 |
| 格式 | 规范统一 |
| 创新 | 有个人见解 |

---

*使用此检查清单确保调研质量*
